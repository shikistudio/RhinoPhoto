# 犀牛照片工坊 Rhino Photo Studio

面向摄影整理、AI 评审和 Lightroom 工作流的桌面照片工作台。

犀牛照片工坊帮助你把旅行、活动和日常拍摄里的大量照片快速整理成可筛选、可评分、可导出的照片资产。它优先服务本地工作流，同时逐步加入人格化 AI 评审、地图轨迹和批量交付能力。

## 下载

公开测试版准备中。正式安装包会发布在本仓库的 [Releases](https://github.com/shikistudio/RhinoPhoto/releases) 页面。

## 适合谁

- 旅行、活动、人像、日常纪实拍摄后需要快速整理照片的人。
- 希望用 AI 辅助初筛、评分、发现值得保留瞬间的摄影用户。
- 需要把照片继续交给 Lightroom 管理，并保留星标工作流的用户。

## 功能方向

- 本地照片导入、分组浏览与批量整理。
- AI 照片评审、评分分布与整体评审结果。
- RAW、JPG、PNG、WEBP、TIFF、HEIC/HEIF 等常见摄影格式支持。
- Lightroom 星标与 XMP 导出工作流。
- GPS、轨迹、地图查询与旅行照片整理。
- 角色化 AI 评审助手，让不同评审风格参与选片。
